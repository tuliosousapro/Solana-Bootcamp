# Solana Bootcamp

This repository contains developments and learning materials for the Solana Developer Course.

## Stack
- **Blockchain:** Solana
- **Languages:** Rust, TypeScript
- **Frameworks:** Anchor
- **Tools:** Solana CLI, Cargo, Node.js

## Structure
- `assets/`: Course slide decks and PDFs.
- `tasks/`: Task tracking and lessons learned.
- `programs/`: Solana programs (smart contracts).
- `tests/`: Program integration tests.
